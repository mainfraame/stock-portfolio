# stock-portfolio

#### Install
```bash
npm install
```

#### Run Tests
```bash
npm test
```

#### Run Build
```bash
npm run build
```
### To start the project
`npm install`
`npm start`

### Build UI
`npm run build`

### Run All Linting
`npm run lint`

### Run UI Linting
`npm run lint:ui`

### Run Server Linting
`npm run lint:server`

### Run All Unit Tests
`npm test`

### Run UI Unit Tests
`npm run test:ui`

### Run All Linting, UI Build, & Start Server
`npm start`
